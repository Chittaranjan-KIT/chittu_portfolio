Projects

Low Earth Orbiting Satellite Network KPI Tool

+

Developed a performance monitoring tool during the testing phase to evaluate key network performance indicators such as latency, jitter, packet loss, and TCP/UDP throughput for LEO satellite communication networks across multiple cities. Raspberry Pi based probe devices were deployed at different geographical locations to simulate end-user environments and collect real-time network performance data. The collected KPI data was visualized using Grafana dashboards to enable real-time monitoring and comparative performance analysis across locations.

Botnet Prediction using Machine Learning (Bachelor's Thesis)

+

This project focused on detecting potential botnet attacks in network traffic by using machine learning techniques to identify unusual communication behavior. Network traffic datasets were analyzed using supervised machine learning algorithms, involving data preprocessing, cleaning, and feature extraction to capture traffic characteristics such as connection frequency, packet size, and communication patterns. A predictive classification model was developed to differentiate between normal and suspicious network activity. Performance was evaluated using accuracy and precision metrics.
