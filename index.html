Ahmed Body - External Aerodynamics

End-to-end steady-state RANS simulation of external flow over a standard Ahmed body at 40 m/s using OpenFOAM.

+
Project Description

Executed an end-to-end steady-state RANS simulation of external flow over a standard Ahmed body at 40 m/s using OpenFOAM. Generated a 3.35M-cell hex-dominant mesh via blockMesh and snappyHexMesh with multi-level surface refinement. Solved using simpleFoam with k-omega SST turbulence model and second-order upwind discretization. Post-processed in ParaView to extract pressure fields, velocity contours, streamlines, and turbulent wake structures.

Key Parameters
  • Freestream velocity: 40 m/s  |  Re ≈ 2.75 × 10⁶
  • Turbulence model: k-omega SST
  • Mesh: 3.35M cells, hex-dominant, multi-level refinement
  • Solver: simpleFoam, SIMPLE scheme, 500 iterations
Results
  • Front stagnation zone with high Cp identified
  • Flow acceleration over roof, separation at rear slant
  • Wake recirculation and velocity deficit downstream
  • TKE concentration at stagnation and separation regions

Frontal Impact Test Simulation - 2012 Toyota Camry

US NCAP 56 km/h frontal rigid wall crash test using the validated CCSA/FHWA FE model (2.25M elements) in Ansys LS-Dyna.

+
Project Description

NCAP full-car frontal rigid wall crash test simulation based on the US NCAP 56 km/h impact test using the validated CCSA/FHWA 2012 Toyota Camry FE model (2.25M elements). Prepared the LS-Dyna keyword file covering contact definitions, rigid wall setup, initial velocity conditions, and termination control. Executed explicit dynamics solver at steady state with tstep = 7×10⁻⁷ s and 0.33% mass reduction. Post-processing performed in LS-PrePost v4.12.6.

Key Parameters
  • Impact velocity: 56 km/h (US NCAP standard)
  • FE model: 2.25M elements, CCSA/FHWA 2012 Toyota Camry
  • Time step: 7×10⁻⁷ s | Mass scaling: 0.33%
  • Post-processing: LS-PrePost v4.12.6 | Filter: CFC60 per SAE J211
Key Results
  • Von Mises Stress Maximum: 261.8 MPa
  • Max Force on Wall: 1,150 kN @ 30 ms
  • Dual-peaked force curve captured: bumper crush + engine collision
  • Full kinetic energy absorbed internally within 80 ms
  • Hourglass energy: <1% - Energy check passed
  • Deceleration data (CFC60) collected per SAE J211

Path Planning & Optimised Racing Line - Suzuka F1 Circuit

Two-phase autonomous racing simulation using real F1 track boundary data from TU Munich.

+
Project Description

The Suzuka Circuit is one of the most challenging Formula 1 circuits in the world - famous for its extremely quick corners, figure-eight layout, and absolute lack of margin for error. This makes it an ideal testbed for autonomous path planning. This project simulates a full two-lap autonomous racing pipeline: the car blindly maps the track during Lap 1 using only local sensor data, then races an optimised minimum-curvature line during Lap 2. The result is a fully autonomous system that finds the smoothest and fastest path through every corner with no human input - the same approach used in Formula Student Driverless competitions.

Key Parameters
  • 34 m sensor radius for real-time boundary cone detection
  • 500-point track resampling for optimiser precision
  • 300 gradient descent iterations with early stopping at Δκ < 1×10⁻⁷
  • 1.5 m safety inset from track walls
  • Learning rate 0.01 with Gaussian gradient smoothing
Skills Learned
  • Real-time path planning via Delaunay triangulation
  • Numerical optimisation via gradient descent
  • Sensor simulation and curvature analysis
  • Autonomous mapping and trajectory planning
Tools Used
MATLAB Delaunay Triangulation Gradient Descent TU Munich Dataset VideoWriter

Tribological Behavior of WC-Co Reinforced Al6061 Metal Matrix Composite
St. Joseph's College of Engineering · Final Year Research · Aug 2022 - Mar 2023
+

Developed and characterized a WC-Co reinforced Al6061 metal matrix composite to study tribological and mechanical behavior improvements over base aluminum alloy.

Methodology
  • Stir casting at 750°C, 350 rpm, 15 min · WC-Co reinforcement: 80% · ~37 µm particle size
  • Testing: Tensile, Compression, Impact, Flexural, Hardness, Wear (ASTM standards)
  • Characterization: SEM, Raman Spectroscopy, TGA, Laser Particle Size Analysis, Electrochemical Corrosion
Key Findings
  • WC-Co reinforcement improved mechanical strength and wear resistance
  • Optimized stirring minimized porosity; uniform particle distribution confirmed by SEM
  • Composite shows potential for automotive and aerospace structural applications
